THE SNAKE GAME


- On the start screen the game will start when the user presses enter

- The start screen includes a small image animation moving side to side until the game begins

- After moving from the start screen the user is presented with a character screen to pick a character

- The user presses the up arrow for the top character and the down arrow for the bottom character

- After the user chooses a character that chosen character will be present in the game screen

- The user should move the snake character with arrow keys to overlap the apple image

- Once the snake overlaps the apple the score increases and the apple moves to another position

- After 5 apples are collected, the game ends and the user is prompted to a game over screen

- If the user presses back/delete they are prompted back to the start screen
